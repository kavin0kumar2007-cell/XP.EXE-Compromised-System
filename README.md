
#  Click the Demo and Kill the virus and Recover the system.
# XP.EXE is a small interactive web project that recreates a Windows XP–style desktop environment infected by a virus. The system boots, runs normally for a moment, then begins to glitch and display security alerts.
# I invite you to Help XP.EXE to a virus free system!, :)
# Project Flow:
## 🟦 Windows XP style boot screen
## 🔐 Classic login screen
## 🖥️ Desktop with draggable icons
## 📂 File Explorer simulation
## 🪟 Window system for applications
## 📟 Simple terminal with commands
## ⚠️ virus infection system
## 🛡️ Virus scanner application
## 🧩 Puzzle challenge to remove the virus
## 💻 BSOD restart simulation
## 🔄 System restores after cleanup
